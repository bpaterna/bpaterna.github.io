1. Instalación

A continuación tienes los pasos que debes seguir para instalar correctamente un sitio web con WordPress en tu servidor.

1. Crear la base de datos

WordPress almacena toda la información de tu sitio web en una base de datos. Cuando se instala se crean las tablas principales. A medida que añades Plugins, si es necsario, se crean tablas adicionales. Por tanto, antes de instalar WordPress necesitas tener una base de datos en tu servidor y un usuario que pueda acceder a ella ya que el asistente de instalación te pedirá el nombre de la base de datos, el servidor dónde se aloja, el usuario que tiene acceso a ella y su contraseña. A partir de estos datos, la instalación de WordPress creará las tablas iniciales y WordPress estará listo para incluir contenido.

Si no sabes como crear la base de datos, en el siguiente enlace tienes los pasos a seguir para hacelo: Creando una base de datos para WordPress

 

2. Descargar los archivos y subirlos al servidor

Desde la URL https://es.wordpress.org/latest-es_ES.zip puedes descargar la última versión disponible de WordPress en español. Puedes descargar el archivo en tu ordenador local y luego subirlo al servidor y descomprimirlo allí o descomprimirlo en local y subir los archivos. También puedes optar por descargar el archivo directamente en el servidor y así te evitas tener que subirlo. Elige la opción que más cómoda te resulte. A continuación tienes algunas indicadiones.

  • Si optas por descargar en local y luego subir al servidor, puedes utilizar un cliente FTP como Filezilla para subir los archivos. Aunque puedes utilizar el que prefieras, hay muchos disponibles.
  • Si prefires descargar en el servidor directamente, los comandos wget y unzip te pueden ayudar. $ wget https://es.wordpress.org/latest-es_ES.zip
    $ unzip latest-es_ES.zip     Recuerda que los archivos, una vez descomprimidos, deberán quedar en el directorio raíz de tu alojamiento.

 

Entrega

A través de la plataforma que utilices en clase, entrega las siguientes capturas de pantalla:

1. Base de datos donde se vean claramente las tablas que se han creado al instalar WordPress.

2. Sección de inicio de tu sitio web.

3. Pantalla principal de la adeministración tu sitio web.
